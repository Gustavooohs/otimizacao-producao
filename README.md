#  Sistema de Otimização de Produção

Sistema de decisão baseado em modelagem matemática para maximizar a produção diária em uma fábrica de engrenagens.

##  Descrição

Projeto acadêmico aplicando conceitos de **Pesquisa Operacional** e **Otimização Matemática** para resolver um problema real de alocação de recursos em uma linha de produção.

##  Como Executar

### 1. Instalar Dependências
